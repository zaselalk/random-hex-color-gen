# random-hex-color-gen

Generate random colors easily in JavaScript.

## Installation

```bash
npm i random-hex-color-gen
```

## Usage

```js
const getRandomColor = require("random-hex-color-gen");

// Generate a random color
const color = getRandomColor();
console.log(color); // e.g., #a3e12f
```

## API

### `getRandomColor()`

Returns a random hex color string (e.g., `#ff00aa`).

## Example

```js
const randomColor = require("random-hex-color-gen");
console.log(randomColor()); // #e4b7f0
```
